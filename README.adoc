= asdf-plugins
:toc: auto

Plugin search and filtering for asdf version manager, built with ReScript, Deno, and formally verified components.

== Architecture

* *ReScript* - Type-safe compilation
* *Deno* - Modern runtime (no Node.js)
* *proven* - Formally verified URL validation for plugin repos
* *TEA* - State management

== Development

[source,bash]
----
deno task build
deno task watch
----
