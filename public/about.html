ASDF Plugin Directory

An interactive, searchable directory of plugins for the asdf version manager.

What is asdf?

asdf is a version manager for multiple languages and tools. It provides a single CLI for managing versions of:

  • Programming languages (Node.js, Python, Ruby, Go, Rust, etc.)
  • Development tools (kubectl, terraform, helm, etc.)
  • Databases (PostgreSQL, Redis, MongoDB, etc.)
  • And many more...

Why Use asdf?

One Tool, Many Languages

Instead of using separate version managers:

  • nvm for Node.js
  • rbenv for Ruby
  • pyenv for Python
  • gvm for Go

Just use asdf for everything.

Consistent Interface

```bash

Install a plugin

asdf plugin add nodejs

Install a version

asdf install nodejs latest

Set global version

asdf global nodejs latest

Set local version (per-project)

asdf local nodejs 18.0.0 ```

How to Use

1. Install asdf ```bash git clone https://github.com/asdf-vm/asdf.git ~/.asdf ```

2. Browse plugins on this site

3. Install a plugin ```bash asdf plugin add ```

4. Install a version ```bash asdf install ```

5. Use the version ```bash asdf global ```
